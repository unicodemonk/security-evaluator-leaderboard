## ğŸ“Š Voice Over Recording Checklist

### Before Recording:
- [ ] Find a **quiet room** (no background noise)
- [ ] Use a **good quality microphone** (or phone/laptop if needed)
- [ ] Read through script **3 times** to get comfortable
- [ ] Drink water (avoid milk/dairy before recording)
- [ ] Disable notifications on devices

### During Recording:
- [ ] Record each scene separately (easier to fix mistakes)
- [ ] Leave **2 seconds of silence** at start and end
- [ ] Speak at **75% of normal speed** (clearer for video)
- [ ] Emphasize **bolded words** slightly
- [ ] Pause at `[PAUSE]` markers for **1-2 seconds**
- [ ] Record multiple takes of each section

### After Recording:
- [ ] Listen to each section
- [ ] Re-record any unclear sections
- [ ] Save files as: `voiceover-scene1.wav`, `voiceover-scene2.wav`, etc.
- [ ] Export final version at **48kHz, 192kbps**

## ğŸ™ï¸ Pro Tips

1. **Smile While Speaking:** Makes your voice sound more friendly and engaging
2. **Stand Up:** Better breath support and vocal projection
3. **Use a Pop Filter:** Reduces "p" and "b" sounds (or speak slightly off-axis from mic)
4. **Room Echo?** Record in a closet or under a blanket fort for better acoustics
5. **Breath Sounds:** Edit these out later, or turn slightly away from mic when breathing
6. **Energy Level:** Keep consistent energy throughout - re-record if you sound tired
7. **Time Check:** Record yourself reading one section and time it - adjust pace as needed

### Record with QuickTime (Mac):
```bash

# File > Export As > Audio Only
```

### Record with Terminal (if sox installed):
```bash
sox -d -r 48000 -c 1 voiceover-scene1.wav
# Press Ctrl+C to stop
```

### Clean Up Audio:
```bash
# Remove background noise (if ffmpeg installed)
ffmpeg -i voiceover-raw.wav -af "highpass=f=200,lowpass=f=3000" voiceover-clean.wav

# Normalize volume
ffmpeg -i voiceover-clean.wav -filter:a loudnorm voiceover-normalized.wav
```
